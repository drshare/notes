# Rust

[Rust官网](https://www.rust-lang.org/zh-CN/learn/get-started)

## 安装

### Windows

官网下载可执行文件安装

### MacOs & Linux

```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## 检验是否安装成功

```shell
# 输出rust 版本
rustc --version
```

## 更新

```shell
# 更新 rust 版本
rustup update
```

## 卸载

```shel
# 卸载 rust
rustup self uninstall
```

> rustup 是管理rust 的工具。
