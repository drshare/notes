# CSS盒模型

css 中一切皆盒子，把组成页面的每一个元素都看做一个盒子。

marign 外边距

border 内边距

padding 内边距

| 名称                   | box-sizing  | 总宽度                                     |
| ---------------------- | ----------- | ------------------------------------------ |
| 标准盒模型             | content-box | 总宽度 = width + padding + border + margin |
| IE盒模型（怪异盒模型） | border-box  | 总宽度 = width + margin                    |
